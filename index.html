🔄 Synchronisation PWA → Native Statut

⚠️ Nouveau workflow : Tes outils (Mastercode, Snipercode, etc.) modifient le fichier index.html à la racine.
Clique sur ⚡ Synchro pour copier les fichiers vers docs/ pour la version native.
💡 Copie tous les fichiers (HTML, CSS, JS, assets)
cd Ostad; mkdir docs -Force; Copy-Item -Path .\* -Destination docs\ -Exclude *.git*, android, docs, node_modules -Recurse -Force
💡 mkdir -Force évite les erreurs si le dossier existe déjà.
💡 Les fichiers inutiles (.git, node_modules, etc.) sont automatiquement exclus.
🔶 Après avoir collé la commande dans PowerShell, clique sur 🔍 Vérifier pour valider.
